How Does It Work Technically?

Moviematching powered by machine learning!
The user input is converted from a text into a number called a vector, using something appropriately called a vectorizer.
Something similar is then done to the plotline and/or keywords of every movie in our database,
and using a type of machinelearning called Nearest Neighbors, FlickPick compares the
input vector to each movies' vector and finds the closest matches, or nearest neighbors.

Et voilà!
We have our matches.

What Movies You Do Have?

Currently we have around 250'000 movies in our database,
our database is based on a wide selection of movies found on themoviedb.org.
Ranging from obscure picks like the Soviet musical Снежная свадьба to box-office hits like Avatar
you can be sure to find something new to watch every time!
